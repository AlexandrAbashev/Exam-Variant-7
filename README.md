# Калькулятор стоимости телефонных разговоров

WPF-приложение для расчета стоимости телефонных разговоров по заданным тарифам с автоматической генерацией квитанций в формате Word.

## Описание

Программа позволяет:
- Рассчитывать стоимость телефонных разговоров по двум тарифам
- Отображать сумму к оплате и количество минут сверх нормы
- Генерировать и сохранять квитанции в формате Word с заполнением данных плательщика

## Требования

- **.NET Framework 4.5** или выше
- **Microsoft Word** (для работы функции генерации квитанций)
- **Microsoft Office Interop for Word**

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/AlexandrAbashev/Exam-Variant-7.git
```

2. Откройте решение в Visual Studio:
   - Файл `PhoneBillCalculator.sln`

3. Убедитесь, что файл шаблона `ЧекШаблон.docx` находится в папке с исполняемым файлом

4. Соберите и запустите проект

## Использование

### Расчет стоимости:
1. Выберите тариф:
   - **Тариф 1**: до 200 минут по 0.7 руб/мин, сверх - 1.6 руб/мин
   - **Тариф 2**: до 100 минут по 0.3 руб/мин, сверх - 1.6 руб/мин

2. Введите количество минут разговора

3. Нажмите "Рассчитать стоимость"

### Генерация квитанции:
1. После расчета заполните:
   - ФИО плательщика
   - Адрес плательщика

2. Нажмите "Сформировать квитанцию"

3. **Чеки автоматически сохраняются** в папку с программой в формате:  
   `Чек_номер_дата.docx` (например: `Чек_123456789_25.12.2023.docx`)

## Тестирование

Проект содержит юнит-тесты (UnitTestProject1), которые проверяют:
- Корректность расчета стоимости для обоих тарифов
- Обработку граничных случаев (минуты меньше/больше лимита)
